PNG파일 문서 삽입기

·배경 투명화해서 업로드한 PNG 파일을 PDF에 삽입함.

1-1. PDF 파일 업로드

드래그하여 PDF파일을 업로드하십시오.
여기에 PDF를 드래그하거나 클릭해서 파일 선택
(선택된 파일 없음)

1-2. PNG 파일 업로드

·A4 크기 문서에 해당 PNG를 넣었을 때 실제 출력되는 크기로 반영해드림.
드래그하여 PNG파일을 업로드하십시오.
여기에 PNG를 드래그하거나 클릭해서 파일 선택
(선택된 파일 없음)
DPI (자동)
크기 배율(%)
출력 너비(cm)
출력 높이(cm)
색상 팔레트
직접 선택
·색상 보정은 투명도를 보존하면서 RGB만 바꿉니다. 가장 안 깨지는 방식임.

2. 삽입 위치 기준 설정

시작 페이지
·PNG를 삽입하기 시작할 페이지를 선택합니다.
예: 첫 장 제외면 2

기준 설정 페이지
·해당 페이지에서 위치/회전을 맞추고 “기준 위치 확정하기”를 누르면 전체 페이지에 적용됩니다.
·시작 페이지와 기준 페이지는 서로 독립입니다.

불투명도
0.35~0.6 추천

·기준 위치 확정 전에는 기준 페이지에서만 오버레이가 보입니다.
·회전하기 체크 시, 오버레이에 마우스를 올리면 ↻↺가 뜹니다. 드래그로 회전합니다.

3. 페이지별 위치 상세 설정

페이지 이동
/ -
숫자 입력 후 Enter(또는 포커스 밖) → 이동

수정/삭제된 페이지
(없음)

4. 화면 미리보기

·미리보기에서 오버레이를 드래그해서 옮기거나, 옮기고 싶은 위치를 클릭하면 중심 위치가 바뀝니다.
·기준 확정 이후: 특정 페이지만 위치를 바꾸면 자동으로 “그 페이지만” 저장됩니다.
